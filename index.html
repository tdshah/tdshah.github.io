Uber vs Lyft Interface Analysis

This is an analysis that compares the Uber and Lyft interfaces. I first sent an Uber & Lyft satisfaction questionnaire to roughly 50 people. This helped me gather insight into the overall interface satisfaction as well as potential pain points that could be investigated further via user interviews. I then chose 5 tasks that can be performed on both interfaces, asked a randomized user sample (with counterbalancing) to complete these tasks on each interface, and then gathered specific metrics to compare the performance. After collecting & intepreting the data, I drafted ideas to improve both interfaces.

Crowdfunding Interface Redesign- Prototype

This redesign project aims to see how current crowdfunding interfaces can be better designed. I looked at examples of medical crowdfunding campaigns to understand how I could create 3 distinct user archetypes (types of users). After doing so, I created a wireframe and thought about design rationales including strengths and weaknesses. Overall, the new design allows deeper integration with social media, more direct patient communication, and a more humanized look into the patient's life. My plan for the next step is to implement this front-end utilizing the React framework.
